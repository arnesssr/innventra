# User Guide

## Products
1. **Creating Products**
   - Navigate to Products > Categories
   - Select a category
   - Fill required fields
   - Add images
   - Save as draft or publish

2. **Managing Inventory**
   - Update stock levels
   - Monitor low stock alerts
   - Track stock movements

3. **E-commerce Integration**
   - Configure store settings
   - Map categories
   - Sync products
